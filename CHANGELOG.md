# Changelog

## v1.0.0 — 26 Mei 2026 🎉

### Managam & Vania Wedding Invitation System

First production release. Features:
- Digital invitation with personalized QR ticket
- WhatsApp blast via Fonnte (Managam & Vania tokens)
- RSVP system with seat selection
- Admin dashboard with live stats
- Check-in system with QR scanner
- RSVP persistence across sessions
- Admin WA notifications on RSVP
- PIC proxy support for group invitations
- Tamu Kehormatan (digital-only) support
- Loading overlay during RSVP submit
- Souvenir & lunch box info on QR ticket
- Manual recipient selection for blast
- Sender filters (Managam/Vania)
- Full E2E test coverage (113/113 checks)
